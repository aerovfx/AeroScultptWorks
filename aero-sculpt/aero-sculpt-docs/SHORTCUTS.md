# ⚡ Hệ Thống Phím Tắt AeroSculpt (Shortcuts & Gestures)

Nắm vững hệ thống phím tắt và thao tác cử chỉ sẽ giúp bạn tối ưu hóa tốc độ làm việc trong AeroSculpt. Dưới đây là danh sách toàn bộ các phím tắt quan trọng:

---

## 🎥 1. Thao Tác Điều Khiển Camera

| Phím Tắt / Thao Tác | Chức Năng | Ý Nghĩa |
| :--- | :--- | :--- |
| **Alt + Kéo chuột trái** | Xoay camera (Orbit) | Xoay 360 độ quanh vật thể để quan sát |
| **Shift + Kéo chuột phải** | Di chuyển camera (Pan) | Trượt khung hình sang các hướng |
| **Cuộn chuột (Scroll)** | Thu phóng (Zoom) | Phóng to để sửa chi tiết, thu nhỏ xem tổng thể |

---

## 🖌 2. Điều Khiển Bút Vẽ (Brushes)

| Phím Tắt | Chức Năng |
| :--- | :--- |
| **`[`** | Giảm kích thước cọ vẽ (Draw Size) |
| **`]`** | Tăng kích thước cọ vẽ (Draw Size) |
| **`Shift` (Giữ chặt)** | Đổi nhanh sang cọ **Smooth** (Làm mịn bề mặt) |
| **`Q`** | Chuyển chế độ cọ sang **ZSub** (Khoét/Đào lõm xuống) |
| **`W`** | Chuyển chế độ cọ sang **ZAdd** (Đắp/Kéo lồi lên) |
| **`N`** | Đảo chiều nhanh qua lại giữa ZAdd (+) và ZSub (-) |
| **`Space`** hoặc **`Tab`** | Mở **Pie Menu** tại con trỏ chuột (để đổi cọ và chỉnh size nhanh) |

---

## 🎭 3. Mặt Nạ (Masking) & Nhóm Lưới (Polygroup)

| Phím Tắt / Thao Tác | Chức Năng |
| :--- | :--- |
| **`Ctrl` + Kéo vẽ trên vật thể** | Vẽ vùng mặt nạ (Mask) để bảo vệ vùng đó khỏi cọ vẽ |
| **`Ctrl + Alt` + Kéo vẽ** | Xóa/tẩy vùng mặt nạ (Erase Mask) |
| **`Ctrl` + Kéo chuột trên vùng trống**| **DynaMesh**: Tái cấu trúc lưới bằng thuật toán Voxel |
| **`Ctrl` + Click trên vùng trống** | Đảo ngược vùng chọn mặt nạ (Invert Mask) |
| **`Ctrl + W`** | Nhóm phần lưới đang được mask thành một **Polygroup** mới có màu riêng biệt |
| **`Shift + F`** | Bật/Tắt chế độ **Polyframe** (hiển thị khung dây và các màu Polygroup) |

---

## ↔ 4. Biến Hình (Gizmo 3D) & Các Tiện Ích Khác

| Phím Tắt | Chức Năng |
| :--- | :--- |
| **`G`** | Bật Gizmo chế độ **Move** (Di chuyển vật thể) |
| **`R`** | Bật Gizmo chế độ **Scale** (Thay đổi kích thước) |
| **`E`** | Bật Gizmo chế độ **Rotate** (Xoay vật thể) |
| **`X`** | Bật/Tắt chế độ đối xứng qua trục X (Symmetry) |
| **`D`** | **Subdivide**: Tăng mức độ chia nhỏ lưới (tăng độ mịn) |
| **`Shift + D`** | **UnDivide**: Giảm mức độ chia lưới (quay về mức chi tiết thấp hơn) |
| **`Ctrl + Z`** | Hoàn tác (Undo) |
| **`Ctrl + Y`** hoặc `Ctrl + Shift + Z`| Làm lại (Redo) |
