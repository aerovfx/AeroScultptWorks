# ⚡ Shortcuts & Gestures / Hệ Thống Phím Tắt ABrush

*Read this in: [English](#english) | [Tiếng Việt](#tiếng-việt)*

---

## English

Mastering the shortcut system and gestures will help you optimize your workflow in ABrush. Below is a comprehensive list of essential shortcuts:

### 🎥 1. Camera Controls

| Shortcut / Gesture | Function | Meaning |
| :--- | :--- | :--- |
| **Alt + Left Click Drag** | Orbit Camera | Rotate 360 degrees around the object |
| **Shift + Right Click Drag** | Pan Camera | Slide the viewport in all directions |
| **Mouse Scroll** | Zoom | Zoom in for details, out for the big picture |

### 🖌 2. Brush Controls

| Shortcut | Function |
| :--- | :--- |
| **`[`** | Decrease Draw Size |
| **`]`** | Increase Draw Size |
| **`Shift` (Hold)** | Quick switch to **Smooth** brush |
| **`Q`** | Switch brush mode to **ZSub** (Carve / Subtractive) |
| **`W`** | Switch brush mode to **ZAdd** (Build / Additive) |
| **`N`** | Quickly toggle between ZAdd (+) and ZSub (-) |
| **`Space`** or **`Tab`** | Open **Pie Menu** at cursor (for quick brush/size changes) |

### 🎭 3. Masking & Polygroups

| Shortcut / Gesture | Function |
| :--- | :--- |
| **`Ctrl` + Drag on object** | Draw Mask to protect an area from brushes |
| **`Ctrl + Alt` + Drag** | Erase Mask |
| **`Ctrl` + Drag on empty canvas**| **DynaMesh**: Recompute mesh using Voxel algorithm |
| **`Ctrl` + Click on empty canvas** | Invert Mask selection |
| **`Ctrl + W`** | Group masked mesh into a new **Polygroup** with a unique color |
| **`Shift + F`** | Toggle **Polyframe** (show wireframe and Polygroup colors) |

### ↔ 4. 3D Gizmo (Transform) & Utilities

| Shortcut | Function |
| :--- | :--- |
| **`G`** | Enable Gizmo in **Move** mode (Translate object) |
| **`R`** | Enable Gizmo in **Scale** mode |
| **`E`** | Enable Gizmo in **Rotate** mode |
| **`X`** | Toggle X-axis Symmetry |
| **`D`** | **Subdivide**: Increase mesh division (smoother details) |
| **`Shift + D`** | **UnDivide**: Decrease mesh division (lower detail level) |
| **`Ctrl + Z`** | Undo |
| **`Ctrl + Y`** or `Ctrl + Shift + Z`| Redo |

## Tiếng Việt

Nắm vững hệ thống phím tắt và thao tác cử chỉ sẽ giúp bạn tối ưu hóa tốc độ làm việc trong ABrush. Dưới đây là danh sách toàn bộ các phím tắt quan trọng:

### 🎥 1. Thao Tác Điều Khiển Camera

| Phím Tắt / Thao Tác | Chức Năng | Ý Nghĩa |
| :--- | :--- | :--- |
| **Alt + Kéo chuột trái** | Xoay camera (Orbit) | Xoay 360 độ quanh vật thể để quan sát |
| **Shift + Kéo chuột phải** | Di chuyển camera (Pan) | Trượt khung hình sang các hướng |
| **Cuộn chuột (Scroll)** | Thu phóng (Zoom) | Phóng to để sửa chi tiết, thu nhỏ xem tổng thể |

### 🖌 2. Điều Khiển Bút Vẽ (Brushes)

| Phím Tắt | Chức Năng |
| :--- | :--- |
| **`[`** | Giảm kích thước cọ vẽ (Draw Size) |
| **`]`** | Tăng kích thước cọ vẽ (Draw Size) |
| **`Shift` (Giữ chặt)** | Đổi nhanh sang cọ **Smooth** (Làm mịn bề mặt) |
| **`Q`** | Chuyển chế độ cọ sang **ZSub** (Khoét/Đào lõm xuống) |
| **`W`** | Chuyển chế độ cọ sang **ZAdd** (Đắp/Kéo lồi lên) |
| **`N`** | Đảo chiều nhanh qua lại giữa ZAdd (+) và ZSub (-) |
| **`Space`** hoặc **`Tab`** | Mở **Pie Menu** tại con trỏ chuột (để đổi cọ và chỉnh size nhanh) |

### 🎭 3. Mặt Nạ (Masking) & Nhóm Lưới (Polygroup)

| Phím Tắt / Thao Tác | Chức Năng |
| :--- | :--- |
| **`Ctrl` + Kéo vẽ trên vật thể** | Vẽ vùng mặt nạ (Mask) để bảo vệ vùng đó khỏi cọ vẽ |
| **`Ctrl + Alt` + Kéo vẽ** | Xóa/tẩy vùng mặt nạ (Erase Mask) |
| **`Ctrl` + Kéo chuột trên vùng trống**| **DynaMesh**: Tái cấu trúc lưới bằng thuật toán Voxel |
| **`Ctrl` + Click trên vùng trống** | Đảo ngược vùng chọn mặt nạ (Invert Mask) |
| **`Ctrl + W`** | Nhóm phần lưới đang được mask thành một **Polygroup** mới có màu riêng biệt |
| **`Shift + F`** | Bật/Tắt chế độ **Polyframe** (hiển thị khung dây và các màu Polygroup) |

### ↔ 4. Biến Hình (Gizmo 3D) & Các Tiện Ích Khác

| Phím Tắt | Chức Năng |
| :--- | :--- |
| **`G`** | Bật Gizmo chế độ **Move** (Di chuyển vật thể) |
| **`R`** | Bật Gizmo chế độ **Scale** (Thay đổi kích thước) |
| **`E`** | Bật Gizmo chế độ **Rotate** (Xoay vật thể) |
| **`X`** | Bật/Tắt chế độ đối xứng qua trục X (Symmetry) |
| **`D`** | **Subdivide**: Tăng mức độ chia nhỏ lưới (tăng độ mịn) |
| **`Shift + D`** | **UnDivide**: Giảm mức độ chia lưới (quay về mức chi tiết thấp hơn) |
| **`Ctrl + Z`** | Hoàn tác (Undo) |
| **`Ctrl + Y`** hoặc `Ctrl + Shift + Z`| Làm lại (Redo) |
